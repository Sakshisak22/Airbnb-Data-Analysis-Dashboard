# 🏠 Airbnb NYC Data Analysis & Interactive Dashboard

## 📌 Project Overview
This project focuses on analyzing the Airbnb NYC dataset and building an interactive dashboard to extract meaningful business insights. It includes data cleaning, exploratory data analysis, visualization, and a user-friendly dashboard.

---

## 🛠️ Tools & Technologies Used
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Streamlit  
- Plotly  

---

## 📊 Project Workflow

### 🔹 1. Data Analysis (Google Colab)
- Data Cleaning & Preprocessing  
- Handling missing values  
- Exploratory Data Analysis (EDA)  
- Advanced Visualizations:
  - Heatmap  
  - Scatter Plot  
  - WordCloud  
- Feature Engineering  
- Basic Machine Learning Model for price prediction  

---

### 🔹 2. Interactive Dashboard (Streamlit)
- KPI Metrics (Average Price, Listings, Reviews)  
- Price Distribution Analysis  
- Location-wise Pricing Insights  
- Room Type vs Price Analysis  
- Map Visualization of Listings 📍  
- Correlation Heatmap  
- Detailed Analysis Tables:
  - Top Expensive Listings  
  - Most Reviewed Listings  
  - Average Price by Room Type  

---

## 📈 Key Insights
- Manhattan has the highest listing prices 💰  
- Entire home/apartment listings are more expensive 🏠  
- Listings with more reviews indicate higher demand ⭐  
- Location plays a major role in pricing 📍
